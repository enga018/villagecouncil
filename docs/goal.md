# Project Goal

Village Council is a production-grade Village Council Management System.

## Current Priority

The current priority is the MVP, with only these core workflows in scope:

- Property Survey
- Household Register

## Later Phases

These will come after the MVP:

- Tax Assessment
- Tax Collection
- Reports
- Super Admin expansion

## Core Principle

The app is designed for field workers first.

Every decision should:

- minimize taps
- reduce typing
- preserve evidence through Property ID, GPS, and photo capture
- allow incomplete surveys to be completed later without losing data
- keep the interface familiar for Village Council staff
